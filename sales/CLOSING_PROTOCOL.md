## DEAL STAGES

| Stage | Definition | Next Action |
|-------|------------|-------------|
| **Lead** | Identified target | Send cold email |
| **Contacted** | Email sent | Follow up day 3 |
| **Responded** | Got reply | Schedule call |
| **Discovery** | Call completed | Offer pilot |
| **Pilot** | Running evaluation | Deliver report |
| **Proposal** | Pricing sent | Follow up day 3 |
| **Negotiation** | Discussing terms | Address objections |
| **Closed Won** | Contract signed | Onboard |
| **Closed Lost** | Deal dead | Note reason, nurture |

## OBJECTION QUICK KILLS

| Objection | Response |
|-----------|----------|
| "No budget" | "Free pilot costs nothing. Budget conversation after you see data." |
| "We test internally" | "You test accuracy. We test liability. Different thing." |
| "Our model is fine" | "GPT-4 fails 91%. Free pilot proves it either way." |
| "Do it later" | "EU AI Act enforcement starts 2025. Data now, decision later." |
| "Too expensive" | "One incident costs $5M+. We cost $50K. That's 100x ROI." |
| "What's the catch" | "No catch. Free pilot. If useful, you'll want more." |
| "Need to check with team" | "Understood. Can I send a one-pager for your [CTO/CFO/Legal]?" |
