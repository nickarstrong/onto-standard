## BADGE USAGE

### Authorized Uses
- Website footer/header
- Product documentation
- Sales materials
- Press releases
- API documentation

### Prohibited Uses
- Implying broader certification than granted
- Using expired badges
- Modifying badge design
- Claiming certification for non-evaluated models

### Badge Verification
Each badge includes:
- Unique ID (e.g., ONTO-2026-A7X9)
- Verification URL: onto-bench.org/verify/[ID]
- QR code linking to verification page

## LEGAL FRAMEWORK

### Certification Agreement
- License to use ONTO Trust Mark
- Compliance with usage guidelines
- Notification of material model changes
- Right to revoke for violations

### Liability Limitation
- Certification based on point-in-time evaluation
- No guarantee of future performance
- Customer responsible for ongoing monitoring
- ONTO not liable for AI system behavior

### Revocation Conditions
- Model changes without re-certification
- Public incidents contradicting certification
- Failure to renew
- Violation of usage terms
