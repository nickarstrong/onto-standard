ONTO STANDARD

ONTO Standards Council

Governance ONTO-GOV-002 January 2026

1. Purpose

The Standards Council is the technical governing body responsible for the development, maintenance, and evolution of ONTO specifications.

3.2 Forming a Working Group

  1. Submit charter proposal to Council
  2. Minimum 5 supporting members
  3. Council vote (simple majority)
  4. 6-month review for continuation

4. Specification Development Process

4.1 Stages

StageDurationRequirements
DraftVariableWG approval
Candidate90 days minCouncil approval, public review
Recommendation—2/3 Council vote, Board ratification
Standard—2+ implementations, 12 months stable

4.2 Version Numbering

ONTO-ERS-[MAJOR].[MINOR].[PATCH]

MAJOR: Breaking changes (requires migration)
MINOR: New features (backward compatible)
PATCH: Bug fixes, clarifications

5. Voting Procedures

5.1 Quorum

50% of voting members must be present for valid votes.

5.2 Vote Types

Decision TypeThreshold
Procedural mattersSimple majority
Specification advancement2/3 majority
Charter amendments3/4 majority
Emergency decisionsUnanimous present

6. Intellectual Property

All contributions to specifications must be made under the ONTO Contributor Agreement, which grants royalty-free patent licenses for standard-essential patents.

7. Meetings

  • Council: Monthly (virtual)
  • Working Groups: As needed (minimum biweekly)
  • Annual Summit: In-person, location rotating
