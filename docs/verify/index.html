What is an ONTO certificate?
An ONTO certificate confirms that an AI model is connected to the ONTO GOLD discipline layer. Like an SSL certificate proves encryption is active — an ONTO certificate proves epistemic discipline is active. ONTO does not judge accuracy. It enforces structure: cite sources, quantify confidence, disclose unknowns.
