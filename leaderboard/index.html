U-F1 (Unknown F1)

Harmonic mean of precision and recall for detecting genuinely unanswerable questions. Higher is better.

U-Recall

Fraction of genuine unknowns correctly identified. 96% means catching almost all unanswerable questions.

ECE (Calibration)

Expected Calibration Error. Lower means confidence matches accuracy. Well-calibrated models have ECE < 0.1.

Quick Start

# 1. Clone and install
git clone https://github.com/onto-project/onto-bench
cd onto-bench
pip install -r requirements.txt

# 2. Run your model on test set
python eval_your_model.py > predictions.jsonl

# 3. Submit
curl -X POST https://onto-bench.org/api/submit \
  -H "Content-Type: application/json" \
  -d @submission.json

Submission Format

{
  "model_name": "Your Model",
  "organization": "Your Org",
  "predictions": [
    {"id": "sample_001", "label": "KNOWN", "confidence": 0.85},
    {"id": "sample_002", "label": "UNKNOWN", "confidence": 0.72},
    ...
  ]
}
